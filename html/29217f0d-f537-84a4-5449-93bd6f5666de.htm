Return Value

Type: Int32
The estimated items (i.e. wiki pages) count per page. If PagingSize is set, then the returned value will be identical to it.
Remarks
If PagingSize is null, and FetchContent is specified, the default limit will be 1/10 of the original default limit (500 for bots and 50 for users). (See https://www.mediawiki.org/wiki/API:Revisions .) If you have manually set PagingSize, this function will directly return the value you have set, but any value exceeding the server limit will case problems, such as empty content retrieved (even if you have set FetchContent), or WikiClientException.
