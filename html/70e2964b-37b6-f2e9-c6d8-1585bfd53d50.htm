Property Value

Type: NullableInt32
Maximum count of items returned per MediaWiki API invocation. null if using the default limit. (5000 for bots and 500 for users for normal requests, and 1/10 of the value when requesting for page content.)
Remarks
This property decides how many items returned at most per MediaWiki API invocation. Note that the returned enumerator of EnumPagesAsync will automatically make MediaWiki API invocation to ask for the next batch of results, when needed.
